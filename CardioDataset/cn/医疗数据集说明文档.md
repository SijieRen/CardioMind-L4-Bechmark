# 医疗数据集说明文档

## 1. 文档概述

本文档详细说明医疗数据集的结构及字段含义。该数据集包含患者原始病历、医生金标准诊断结果、模型输入问题。



## 2. 数据结构总览

每条数据为一个完整的 JSON 对象，包含以下核心部分

| 字段名    | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| seq_id    | 数据唯一标识符，用于数据溯源                                 |
| raw_query | 患者原始病历全文，包含主诉、现病史、既往史、个人史、家族史、体格检查、辅助检查等完整临床信息 |
| ans_label | 医生根据患者病历给出的权威诊断结果，作为评估模型表现的金标准。 |
| question  | 模型输入问题，由患者原始病历和标准化诊断任务提示词组成，明确要求模型按照临床推理流程完成诊断 |
